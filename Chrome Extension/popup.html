About

BitCrystals box is an open source wallet built for the BitCrystals (www.bitcrystals.com used in Moonga and Spels of Genesis

Powered by APIs from Insight.io, Chain.so, and Counterpartychain.io

Source code is available on Github at
https://github.com/everdreamsoft/bitcrystalsbox

Based on the work of LTB network wallet https://github.com/loon3/LTB-Companion-Wallet

Please report any bugs or issues to the Issues section of the Github repository
